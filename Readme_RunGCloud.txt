Steps to do HW0-programming part on Google Cloud:

1. Set up the environment as listed in the "F19_GCloudSetup.pdf"
    1.1. Project: NLP-0
    1.2. SSH keys (in Compute Engine/Metadata, may outdate)
    1.3. Create VM instance
2. Run project
    2.1. Start the instance
    2.2. Click SSH to start the client
    2.3. Click the wheel in the upper right corner of the SSH client and upload/download file.
    2.4. Run scripts.
    2.5. Close the client and STOP the instance.
